# FORSA-LIYA

AI-powered developer learning platform that helps users learn by building real-world projects, solving quizzes, and getting intelligent feedback on their code.

---

## Concept

FORSA-LIYA is an interactive learning platform designed for developers.

Instead of passive learning, users:
- Build real projects
- Get AI-generated challenges
- Take quizzes
- Write code directly in the browser
- Receive AI feedback on their code

---

## Features

### Authentication
- Laravel Breeze
- Register / Login / Logout

### Learning Tracks
- Frontend / Backend / Fullstack
- Branch-based progression
- Difficulty levels:
  - Beginner
  - Intermediate
  - Advanced

### AI Project Generator
- Generates real-world projects based on:
  - Track
  - Branch
  - Level
- Fallback system when AI fails

### Quiz System
- Auto-generated questions (AI or fallback)
- Multiple choice + scenario questions
- Score calculation

### Code Editor
- Monaco Editor integration
- Supports:
  - JavaScript
  - HTML
  - PHP
- Real-time typing experience

### AI Code Review
- Evaluates user code
- Returns:
  - Score (/20)
  - Feedback
  - Mistakes
  - Improvements

### Result System
- Pass / Fail logic
- XP system
- User progression tracking

### Dashboard
- Projects history
- Results
- Progress tracking
